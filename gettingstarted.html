Getting Started With NodeJS

So, you are convinced to use NodeJS, great! This page will help you get started.

Posted on: 6/3/2019 by Gabe.

Installation

Before we begin building incredible web applications with NodeJS we are going to need to actually install it onto our computer. The installation is pretty straight forward.


• Step 1: Download the Installer from NodeJS's Website.

• Step 2: Run the installer.

• Step 3: Install NodeJS & NPM.

•Step 4: Restart your machine/system.

• Step 5: Enter your command-line/terminal.

• Step 6: Run node -v and npm -v and things should show up, if nothing shows up or there was an error the installation didn't work.

Your First Line

In this section we will walkthrough the process of creating your first NodeJS file and getting your first line of code written. This section implies that you have the basic knowledge on how your operating system works.


• Step 1: Create a new folder

It doesn't matter where you put this folder as long as it is a place you will remember.


•Step 2: Create a file inside that folder with the .js file extension

The name of the file doesn't matter as long as it has the .js file extension (ex. main.js or index.js).


• Step 3: Enter that file

You will need to open this file with some sort of IDE or plain-text editor. Some include VS-Code, Brackets, Atom, etc.


• Step 4: Your first line of code

Within NodeJS there are a ton of different things you can do but for now we are only going to do something very simple. Copy and paste the following code in your document:


// This will print Hello World to the console.
console.log('Hello World!');

•Step 5: Run your application

So far we have only told our application to print something to the console so it will be very basic but this is just a start. Enter your terminal again and navigate over to the folder you created. After in your folder just type:


node enter_filename_here.js

And vuala! You have built the most basic NodeJS application. Now, go watch some youtube videos and continue to use this brilliant framework!
