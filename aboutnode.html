About NodeJS

This page will explain all about what node is and what it's purpose is.

NodeJS is a javascript runtime environment. In simple terms it is a flexable, lightweight, and built off of Javascript. NodeJS is perfect for creating applications that require front-end and backend as it can handle both. It can handle things like creating, reading, opening, writing, deleting and closing files on a server. Node can collect form data. Most usefull, node can add, delete, or modify data within a database.
