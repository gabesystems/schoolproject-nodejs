About NodeJS

This page will display the history of NodeJS, the uses of NodeJS, and a breif overview on what it is.

What is NodeJS?

NodeJS is a free and open-sourced Javascript runtime environment. Unlike standard Javascript, NodeJS gives you everything to run the program outside of a browser. Meaning you can create or run backend and server-side applications with NodeJS. But, you can also use NodeJS if you want to create only front-end material also. NodeJS offers a bunch of frameworks or plugins that allow you to easily install different packages to help build your application. For example, there is vue.js that makes creating single page applications and user interfaces very easy. Source here

NodeJS's History

NodeJS was written initially in 2009 by Ryan Dahl. When it was originally released the environment was only supported on Mac OS and Linux but later on, it became supported on Windows. Node Package Manager (NPM) was released a year later in 2010 which allowed people to download, install, uninstall, and update NodeJS libraries. In late 2011, Microsoft implemented their native version of NodeJS for windows. Source here

How Can NodeJS Be Used?

NodeJS is great for building front-end and backend applications. This means you can run server requests while also building amazing user-interfaces (UI). Unlike rails or ruby where they are only server-side languages, NodeJS offers the best of both worlds. Some common uses for NodeJS are as follows:

  • Browser Games
  • Chat Rooms
  • Collecting Data
  • Streaming
  • Linux/Daemon Management
  • Panels
